### 개발환경 세팅
##### .env 세팅
```
NOTION_TOKEN=<Notion Api Key>
NOTION_ABOUTME_PAGE_ID=<소개페이지 ID>
NEXT_PUBLIC_SITE_URL=<포트폴리오 링크>
NOTION_DATABASE_ID=<데이터베이스 ID>
```

##### npm install
``` 
npm install --legacy-peer-deps
```

##### npm start
```
npm run dev
```
